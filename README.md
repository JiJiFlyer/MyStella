# 数据可视化项目
后台代码为VB的WEB窗体。 <br>
实现公司财务数据的可视化，帮助管理者纵览公司情况，辅助决策。<br>
# 所用技术
asp.net WebForm+sqlserver+vb.net+js+css+echart+bmap（百度地图API）。
